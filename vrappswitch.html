Magician's Hat Application Switcher

The Magician's Hat was a project for my Human-Computer Interaction class at UC Santa Cruz, finished over the span of ten weeks. Our team of four decided to create a working prototype for an application switcher for the Oculus Rift DK2, which would allow users to use voice commands to exit the game and load another without taking off the headset.

Below is a copy of our final report, which involves a full HCI/UX design cycle, including low/high-fidelity prototypes, storyboarding, user testing, user research, competitive analysis, personas, IDEOmethod application, requirements gathering, etc.

  • Roles: User experience, low/high-fidelity prototypes, storyboarding, user testing, user research, competitive analysis, personas, requirements gathering, IDEOmethods, 3D modeling, participant setup, running experiments
